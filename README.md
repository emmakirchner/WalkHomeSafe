# \# 🚶‍♂️ WalkHomeSafe

# 

# \*\*Your personal safety companion for getting home safely\*\*

# 

# \---

# 

# \## 🎯 Purpose \& Goal

# 

# WalkHomeSafe is a native Android app (Android 9.0+) designed to support a safe journey home and enhance user safety during nighttime or solo walks.

# 

# \---

# 

# \## 💡 Benefits

# 

# \* Increased sense of security

# \* Faster emergency response

# \* Community involvement

# 

# \---

# 

# \## ⚙️ Core Features

# 

# \* Setting emergency contacts with automated messages

# \* Location sharing with selected contacts

# \* Community reporting of unsafe areas

# \* ...

#

#
